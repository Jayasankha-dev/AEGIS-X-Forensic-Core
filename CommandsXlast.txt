51. Master File Table (MFT) & USN Journal Analysis
Command: fsutil usn readjournal C: csv
Description: Reads the low-level "Journal" of every single file change ever made on the disk.
Objective: Even if a hacker deletes a file and clears the Event Logs, the USN Journal (built into the NTFS file system) keeps a record of the deletion. This is the ultimate "History" of the hard drive.

52. Nuclear Process Termination (Suspension First)
Command: Get-Process | Where-Object { $_.CPU -gt 50 } | ForEach-Object { $_.Suspend() }
Description: Instead of killing a process (which might trigger a "Watchdog" to restart it), this Freezes the process in RAM.
Objective: Stops the virus from acting without letting it know it's being attacked. You can then analyze its memory while it's "frozen."

53. Removing "Image File Execution Options" (The Silent Hijack)
Command: Get-ChildItem "HKLM:\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Image File Execution Options"
Description: Checks if a hacker has redirected common apps.
Objective: Hackers use this so that when you try to open taskmgr.exe or regedit.exe, it actually opens their virus instead. Delete any key here that matches a system app.

54. Total Network Stack Purge (The 2100 Standard)
Command: ```powershell
netsh branchcache reset; netsh int ipv6 reset; netsh advfirewall reset; nbtstat -R; nbtstat -RR

**Description:** Resets every communication protocol in the OS.
**Objective:** Clears hidden "Static Routes" and "DNS Poisoning" that hackers use to redirect your traffic even after a reboot.
